서울잠실야구장 주차장 정보를 개선할거야
상황: 서울 열린데이터 광장의 '서울시 시영주차장 실시간 주차대수 정보' API를 활용해 
잠실종합운동장의 주차 혼잡도를 판단하는 백엔드 로직을 작성해야 해.

데이터 소스:

- 필수 필드: PKNM (주차장명), QUE_STATUS (현재 주차 상태: 10-여유, 20-보통, 30-혼잡), CUR_PK_CNT (현재 주차 대수), TP_PK_CNT (전체 주차 면수)

요구사항:
2. 기존의 '네이버 블로그 검색 기반 점수 로직'을 완전히 제거하고, 위 API 결과값을 바탕으로 'bad', 'normal', 'good' 상태를 반환하도록 수정해줘.
3. 주차장명이 '잠실'을 포함하는 데이터만 필터링해서 평균 혼잡도를 계산하거나, 가장 대표적인 잠실종합운동장 주차장 데이터를 추출해줘.
4. 예외 처리: API 호출 실패나 데이터가 없을 경우 'unknown'을 반환하고 로그를 남겨야 해.
5. Flutter 앱에서 이 데이터를 효율적으로 렌더링할 수 있도록 JSON 응답 구조를 설계해줘.

특이사항: 
- 이전에 사용하던 '블로그 키워드 카운팅' 방식은 부정확해서 폐기할 거야.
